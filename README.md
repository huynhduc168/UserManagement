# UserManagement

- python version
python 3.13.3
- pip version
python 25.1.1

- tạo môi trường ảo
    python -m venv venv

- chạy môi trường ảo
    source venv/Scripts/activate

- cài FastAPI và Uvicorn
    pip install fastapi uvicorn

- chạy server
    uvicorn app.main:app --reload

- Dừng server & thoát môi trường ảo
    + Dừng server: nhấn Ctrl + C
    + Thoát môi trường ảo: deactivate
